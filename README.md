## Online SRET task 

Created for the dissertation project entitled "Depression and the Self: A Longitudinal Examination of Biased Self-Schemas and Depression Symptom Dimensions" by Rachel Weisenburger of the Mood Disorders Lab at The University of Texas at Austin.

This task has been adapted via the programming and instructions from Dr. Justin Dainer-Best of the Bard College Affective Science Lab, https://affectlab.bard.edu.
